# INFERNOLab

A cinematic developer portfolio built with modern web technologies, immersive animations, and interactive 3D experiences.

## Live Demo

Frontend: infernolab.vercel.app
Backend API: infernolab.onrender.com

---

# Overview

INFERNOLab is a high-performance full stack portfolio designed to showcase:

* Full Stack Development
* AI / ML Engineering
* Distributed Systems
* Interactive 3D Web Experiences
* Modern UI Animation Systems

The project combines:

* React + TypeScript
* TailwindCSS
* Framer Motion
* Three.js / React Three Fiber
* Express Backend
* Resend Email API

with a fully custom dark lava-inspired cinematic design system.

# Installation

## Clone Repository

```bash
git clone https://github.com/Choudhary-Sahil099/INFERNOLab.git
```

---

# Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# Environment Variables

Create a `.env` file inside backend:

```env
RESEND_API_KEY=your_resend_api_key
PORT=5000
```

---

# Production Build

## Frontend

```bash
npm run build
```

## Backend

```bash
npm run build
npm start
```
